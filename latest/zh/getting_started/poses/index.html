    自定义姿势

    本模组允许自定义NPC的各种姿势，输入以下指令： 

    /npc edit pose <姿势ID>

    当实体类型设置为PLAYER时，工作姿势将被设置为

    • 站立（standing）（默认）
    • 游泳（swimming）
    • 鬼鬼祟祟 / 潜行（crouching）
    • 睡觉（sleeping）

    其他的姿势只是更改了名称的显示位置，在视觉上没有任何区别。
